# Ghetto_Botato

A Node.js Typescript twitch bot used at [twitch.tv/ghetto_spud](twitch.tv/ghetto_spud)

After pulling:
    - Install Node.js + NPM
    - `npm install`

To build:
    - `npm run build`

To start:
    - `npm run start`

Reference the sample config files included in repo to configure the bot to connect to a channel.
